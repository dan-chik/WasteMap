# WasteMap

WasteMap is an open-source civic tool for reporting and visualizing waste hotspots in urban areas.

## Installation

1. Clone the repository  
   `git clone https://github.com/dan-chik/WasteMap.git`

2. Install dependencies  
   `npm install`

3. Add your Mapbox API token to `.env`

4. Start the application  
   `npm run dev`
